# Pharmacy Management System

A complete **desktop Pharmacy Management System** built with **C# WPF** and a local **SQLite** database. Developed as a real client project to fully digitize daily pharmacy operations — from sales and purchases to accounts, salaries, and reporting — with no dependency on the internet.

## Key Features

- **Secure Login** — Role-based login window (Admin/Employee access)
- **Dashboard** — At-a-glance summary of daily sales, stock, and business activity
- **Medicines / Inventory** — Add, update, and manage medicine records with barcode support
- **Sales Management** — Create sales, print bills/invoices, and track sale history
- **Purchase Management** — Record purchases from vendors with purchase vouchers
- **Returns Handling** — Manage sales and purchase returns
- **Accounts & Debts** — Track customer/vendor accounts and outstanding debts
- **Employee Management** — Maintain employee records, salaries, and expenses
- **Reports** — Generate detailed sales, purchase, and financial reports
- **Bill / Voucher Printing** — Custom print layouts for bills and purchase vouchers
- **Barcode Generation** — Built-in barcode window for product labeling
- **Activity Logs** — Track system usage and changes
- **Multi-language Support** — Language resource support for localization
- **Settings** — Configurable application settings per business

## Tech Stack

- **Frontend/UI:** C# WPF (XAML)
- **Database:** SQLite (local, offline-first — no internet or server required)
- **Architecture:** Page/Window-based navigation with a dedicated Database Helper layer for data access

## Why This Matters

This project demonstrates practical experience in exactly the kind of system many small businesses need:

- **Master–detail data entry** (e.g., adding a sale/purchase with line items)
- **Search, filter, and reporting** across transactional data
- **Custom print formats** for bills and vouchers
- **Offline desktop operation** with a local database (no cloud dependency)
- **Multi-module business software** — inventory, accounts, employees, and reports in one system
